# SkyblockTexturesBegone

Source code for the SkyblockTexturesBegone Fabric mod.

## Why?
**Hypixel is going to start forcing their ugly ass skyblock texture pack, as they have shown on the alpha, so I made this so I personally don't have to deal with it, decided, why not post it for others to use.**

_Any of your own texture packs will still work._

## Requirements

**Requires Fabric API**

## More Info

_Only version 1.21.11 at the moment, if enough interest, I do plan on providing support for later versions._

## Building

Requires JDK 21.

```
./gradlew build
```

The built jar is in `build/libs/`.
